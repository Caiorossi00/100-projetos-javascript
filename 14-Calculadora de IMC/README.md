Calculadora de IMC desenvolvida em JavaScript com manipulação de DOM. A aplicação permite que o usuário informe peso e altura para calcular automaticamente o Índice de Massa Corporal (IMC).

O sistema valida os valores informados, aplica a fórmula do IMC (peso dividido pela altura ao quadrado) e classifica o resultado de acordo com as faixas padrão utilizadas na avaliação corporal. Após o cálculo, o valor do IMC e sua classificação são exibidos dinamicamente na interface.

## MVP

- Entrada de peso e altura pelo usuário
- Validação dos campos antes do cálculo
- Cálculo automático do IMC
- Classificação do resultado (magreza, normal, sobrepeso, obesidade)
- Exibição dinâmica do resultado na interface
