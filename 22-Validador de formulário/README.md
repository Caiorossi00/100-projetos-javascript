Validador de login simples desenvolvido em JavaScript com manipulação de DOM. A aplicação simula a autenticação de um usuário comparando os dados inseridos em um formulário com informações previamente armazenadas em memória.

O sistema exibe inicialmente os dados do usuário cadastrado e, ao tentar realizar o login, valida se os campos foram preenchidos corretamente. Em seguida, compara nome, email e senha com os dados existentes, retornando mensagens específicas para cada erro ou confirmando o acesso quando todas as informações estão corretas.

## MVP

- Exibição de dados de usuário armazenados
- Formulário com campos de nome, email e senha
- Validação para garantir que todos os campos foram preenchidos
- Verificação individual de cada dado informado
- Mensagens de erro específicas para cada campo incorreto
- Confirmação de login quando os dados estão corretos
