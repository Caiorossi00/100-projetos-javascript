Gerador de frases motivacionais desenvolvido em JavaScript com manipulação de DOM. A aplicação seleciona aleatoriamente uma frase de uma lista pré-definida e exibe na interface ao usuário.

O sistema utiliza geração de números aleatórios para escolher uma frase da lista, armazenando-a em uma variável e atualizando dinamicamente o conteúdo de um elemento HTML para exibição. Cada execução gera uma nova frase, oferecendo variedade e inspiração a cada interação.

## MVP

- Lista de frases motivacionais pré-definidas
- Seleção aleatória de uma frase a cada execução
- Armazenamento da frase selecionada em uma variável
- Exibição dinâmica da frase na interface
- Atualização da frase ao acionar a função
