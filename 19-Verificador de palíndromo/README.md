Verificador de palíndromos desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o usuário insira uma palavra e verifica se ela pode ser lida da mesma forma de trás para frente.

O sistema transforma a palavra informada em minúsculas, inverte sua ordem utilizando métodos de array e compara o valor original com sua versão invertida. Com base nessa comparação, exibe dinamicamente na interface se a palavra é ou não um palíndromo.

## MVP

- Entrada de texto pelo usuário
- Normalização da palavra para comparação
- Inversão da string utilizando métodos de array
- Comparação entre palavra original e invertida
- Exibição do resultado na interface
