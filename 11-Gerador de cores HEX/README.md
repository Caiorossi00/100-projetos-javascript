Gerador de cores aleatórias em formato hexadecimal desenvolvido em JavaScript com manipulação de DOM. A aplicação cria uma cor hexadecimal aleatória composta por seis caracteres selecionados a partir de dígitos e letras válidos para esse formato.

O algoritmo utiliza um laço de repetição para selecionar caracteres aleatórios da sequência hexadecimal (`0–9` e `A–F`) e construir a cor no padrão `#RRGGBB`. Após a geração, o valor da cor é exibido dinamicamente na interface.

## MVP

- Geração de cor aleatória em formato hexadecimal
- Algoritmo de construção de cor no padrão `#RRGGBB`
- Uso de caracteres válidos do sistema hexadecimal
- Exibição dinâmica da cor gerada na interface
