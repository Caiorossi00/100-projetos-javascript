Gerador de tabuada desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o usuário insira um número e visualize automaticamente sua tabuada de 1 a 10.

O sistema valida a entrada para garantir que o valor seja numérico, percorre um laço de repetição para calcular cada multiplicação e constrói dinamicamente o resultado. A tabuada é exibida na interface utilizando formatação com quebras de linha.

## MVP

- Entrada de número pelo usuário
- Validação para garantir valor numérico
- Geração da tabuada de 1 a 10
- Cálculo automático das multiplicações
- Exibição formatada do resultado na interface
