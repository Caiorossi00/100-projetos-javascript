Conversor de moedas desenvolvido em JavaScript com manipulação de DOM. A aplicação permite converter valores entre diferentes moedas a partir de taxas de câmbio pré-definidas.

O sistema captura o valor informado pelo usuário, identifica as moedas de origem e destino selecionadas e realiza a conversão utilizando o dólar como base intermediária. Após o cálculo, o resultado é exibido dinamicamente na interface com formatação de duas casas decimais.

A aplicação também valida a entrada para garantir que o valor informado seja numérico antes de realizar a conversão.

## MVP

- Entrada de valor para conversão
- Seleção de moeda de origem e destino
- Conversão baseada em taxas pré-definidas
- Uso de moeda base para cálculo (USD)
- Validação de entrada do usuário
- Exibição dinâmica do resultado na interface
