Validador de senha em tempo real desenvolvido em JavaScript com manipulação de DOM. A aplicação monitora o campo de senha enquanto o usuário digita e verifica automaticamente se a senha atende aos critérios de segurança definidos.

Cada requisito é validado individualmente utilizando expressões regulares e verificação de tamanho. Conforme o usuário digita, a interface atualiza dinamicamente o estado de cada regra, indicando quais requisitos já foram atendidos e quais ainda faltam.

## MVP

- Campo de entrada para digitação de senha
- Validação em tempo real conforme o usuário digita
- Verificação de tamanho mínimo da senha
- Verificação de presença de números
- Verificação de presença de letra maiúscula
- Verificação de presença de símbolo
- Atualização visual das regras atendidas e pendentes
