Quiz interativo desenvolvido em JavaScript com manipulação de DOM. A aplicação apresenta uma sequência de perguntas com múltiplas opções de resposta e calcula a pontuação do usuário ao final.

O sistema percorre um array de perguntas, renderizando dinamicamente cada questão e suas opções na interface. Ao selecionar uma resposta, o sistema verifica se está correta, atualiza a pontuação e avança para a próxima pergunta. Quando todas as perguntas são respondidas, o resultado final é exibido.

O fluxo é controlado por um índice que representa a pergunta atual e uma variável de pontuação acumulativa.

## MVP

- Lista de perguntas com múltiplas opções
- Renderização dinâmica das perguntas
- Seleção de resposta via botão
- Verificação de resposta correta
- Controle de progresso entre perguntas
- Cálculo de pontuação
- Exibição do resultado final
