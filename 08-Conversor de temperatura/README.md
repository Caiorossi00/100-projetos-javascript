Conversor de temperatura desenvolvido em JavaScript com manipulação de DOM. A aplicação permite converter valores entre as escalas Celsius e Fahrenheit através de entradas fornecidas pelo usuário.

O sistema valida os valores informados, aplica a fórmula matemática correspondente à conversão escolhida e exibe o resultado formatado na interface. O valor convertido é apresentado com duas casas decimais para melhorar a legibilidade.

## MVP

- Conversão de Celsius para Fahrenheit
- Conversão de Fahrenheit para Celsius
- Validação de entrada numérica
- Aplicação correta das fórmulas de conversão
- Exibição do resultado formatado na interface
